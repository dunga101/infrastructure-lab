# pfSense Enterprise Lab Firewall

This folder tracks the evolution of an enterprise-style firewall and routing lab built using pfSense on Hyper-V.

pfSense acts as the single routing, NAT, and security boundary for all internal lab networks.

## Documented Phases
- Phase 2: Core Network Baseline
- Phase 3: VLAN Segmentation (planned)
- Phase 4: Firewall Hardening (planned)
- Phase 5: IDS/IPS and Logging (planned)
- Phase 6: Hybrid Connectivity to Azure (planned)
